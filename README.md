# WindFarmSpotter-Python for Web 





## Build Instructions 

Clone this repo:
```
git clone -b wfs-py-web https://github.com/codeamt/WindFarmSpotter WindFarmSpotter 
cd WindFarmSpotter
```

Install packages:
```
pip3 -r install requirements.txt 
```


## Running 

From the root of the repo:
```
python3 run.py
```
